STS Guide

This guidebook should only be used for STS purposes and only kept within management or above staff members. Failing to follow this guidebook will and can result in staff action. Aswell as leaking this guidebook will result in a staff blacklisted that will not be appealable.

To complete a STS you may follow this simple guide.

Before you start the STS please run the following commands:


**LOAD LAYOUT COMMAND HERE**
:m We are sorry to disturb your roleplays! An STS will be occurring due to unforeseen unprofessional roleplays. Please wait to be teleported.
:m An STS is occurring! Please step out of your vehicles immediately and wait until you have been teleported. Please then make your way to the yellow line.

Then once you have alerted people imediatly run the following commands:


:bring all
:prty 500
:pt 500

You may need to do also :heal all 3 times once people have been teleported


When you start the STS please run this command:


:m Before we conduct this STS, we will be ensuring everyone’s character is realistic! Staff will come around and check your avatar. In the meantime do not communicate with other players!

When you have done the above commands please complete the following commands to show the server rules:


:m Make sure to roleplay all crashes. Only time you can avoid it if it's de-sync or under 15 mph. Anything else is prohibited.
:m Do not use any authorised firearm on the police team unless trained. You can apply via our communications server.
:m When a moderator has their sirens on you are required to pull over.
:m Do not use any guns on the civilian without any permission from a moderator. This is to keep roleplays realistic! Failing to do so will result in a kick as this is classed as staff evasion!
:m Despite the fact that we are a UK based server. Please ensure you are driving via map layout (right hand side)!
:m When in a moderator scene, you are required to stay at the scene. If you are seen running away from the scene you will be kicked!
:m Finally please keep roleplays realistic. We are always aiming for a realistic roleplay!
:m Please be prepare to be loaded.

Once you have done the STS please run the following Commands:


:load all

x3

:prty 0
:pt 0
**UNLOAD LAYOUT COMMAND HERE**
