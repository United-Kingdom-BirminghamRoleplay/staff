Message Preset Guide

When actioning someone or responding to a message, you're required to use the formats provided below as a foundation, which you can expand upon; however, we strongly recommend having any modifications double-checked by our high-ranking team to ensure accuracy and consistency.

Message Presets

ACTION MESSAGES

Your are being warned for [REASON]. Please take note your warnings reset after every session.
You are being kicked for [REASON]. Please do not rejoin back within [XX] minutes. Please feel free to appeal this via the communications server using the code: UKBRUM.
You are being banned for [REASON]. Please feel free to appeal this via the communications server using the code: UKBRUM.

MOD CALL MESSAGES

Greetings! I am [NAME], how may I assist you today?
Is there anything else that I can help you with today?
Feel free to leave a staff review on the communications server via the code: UKBRUM.
